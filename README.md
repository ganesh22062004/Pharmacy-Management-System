# 💊 Pharmacy Management System

A simple and interactive **Pharmacy Management System** built using **Python**. This project helps manage medicine inventory, generate customer bills, and monitor stock through a menu-driven command-line interface.

---

## 📌 Features

- ➕ Add new medicines
- 📦 Update existing medicine stock
- 💳 Generate customer bills
- 📋 View available medicine stock
- ⚠️ Low stock alerts
- 🚪 Exit the application

---

## 🛠️ Technologies Used

- **Python 3**
- Dictionary Data Structure
- Command-Line Interface (CLI)

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or later

### Installation

1. Clone the repository:

```bash
git clone https://github.com/ganesh22062004/Pharmacy-Management-System.git
```

2. Navigate to the project directory:

```bash
cd Pharmacy-Management-System
```

3. Run the application:

```bash
python pharmacy_management_system.py
```

---

## 📖 How It Works

After running the program, the following menu is displayed:

```text
===== Pharmacy Management System =====

1. Add Medicine
2. Billing
3. View Stock
4. Exit
```

Choose an option by entering the corresponding number and follow the prompts.
